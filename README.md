# ComfyUI Error Fixer Plugin

自动捕获ComfyUI工作流错误并提供修复建议的插件。

## 安装方法

1. 将此项目下载到 `ComfyUI/custom_nodes/` 目录下
2. 重启 ComfyUI
3. 插件会自动加载并开始工作

## 功能特点

- 🔍 自动捕获工作流运行错误
- 🔧 在错误节点上显示修复按钮
- 📋 错误对话框中自动添加解决方案按钮
- 🌐 一键跳转到解决方案网站
- 🔄 完全非侵入式设计

## 使用方法

1. 运行工作流时如果出现错误，插件会自动捕获
2. 点击错误对话框中的"Error Fixer Online"按钮
3. 或点击节点右上角的🔧图标
4. 自动跳转到解决方案网站
